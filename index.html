Minimizing the Internet

Minimalism. Thats what this webpage represents. Webpage, not website. This is not a site of multiple pages, but a single html file using sections to divide itself into "pages". This adheres to the foundational commandments of both the "Small Internet Movement", and the "Ten Commandments of HTML".


FOSS and the Church of Emacs

As stated by the all holy Saint IGNUcius, "There is no system but GNU, and Linux is one of its kernels.", the only true operating systems are those following the FOSS philosophy. As a continuation of this idea, all projects created and maintained by me, Alex Bell, will adhere to the prinicples of FOSS. In terms of this page itself, it contains only html and css, with no java script that requires your machine to process. 100% of the processing power is done on the hosting side, with your machine's only task being to request and disply this page -- thus being completely libre.
